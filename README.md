## Installation 

Install all dependencies 
```sh 
yarn 
```
## Run project
```sh
yarn start
```

## Tests

Run tests
```sh
yarn test
```

## Storybook
Run storybook
```sh
yarn storybook
```

## Additional info
Remember to run REST API back-end from this link https://github.com/netguru/heroes-api/
